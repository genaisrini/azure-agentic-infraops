## Executive Summary

The **agent-testing** infrastructure provides a comprehensive Azure environment for validating
the agentic infrastructure workflow. The architecture prioritizes:

1. **Cost Efficiency** - Basic/consumption SKUs appropriate for development
2. **Completeness** - All major Azure service categories represented
3. **Compliance** - EU data residency (swedencentral primary region)
4. **Best Practices** - Azure Verified Modules, Azure AD authentication
